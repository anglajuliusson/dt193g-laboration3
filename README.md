# Vue Concerts SPA

En enkel Single-Page Application byggd med Vue 3 som visar konserter jag har varit på.  
Applikationen konsumerar en egen backend (Fastify + MySQL) och låter användaren lägga till och ta bort konserter.

## Funktioner
- Visa alla konserter i en tabell
- Markera om konserten är "sedd" eller inte
- Lägga till nya konserter via formulär
- Ta bort konserter direkt från tabellen
- Komponentbaserad arkitektur med Props och Emits
- Reaktiv data och two-way-binding med Vue

## Teknologier
- Vue 3 + Composition API
- Fastify backend med MySQL
- Fetch API för kommunikation med backend
- Scoped CSS för komponenter
- Single-Page Application med Vue Router
